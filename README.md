# ProyectaDi

ProyectaDi es una plataforma colaborativa de gestión de proyectos orientada a la planeación, seguimiento técnico, control presupuestal, registro de avances, generación de gráficas e informes automáticos.

## Objetivo general

Desarrollar una plataforma intuitiva, colaborativa y escalable que permita organizar, monitorear y reportar el avance técnico, operativo y financiero de proyectos mediante cronogramas, etapas, tareas, presupuesto, evidencias, gráficas e informes automáticos.

## Tecnologías principales

- Backend: FastAPI + Python
- Frontend: React + TypeScript
- Base de datos: PostgreSQL / Supabase
- Reportes: generación de PDF
- App de escritorio futura: Tauri
- Control de versiones: Git + GitHub
