# Diseño inicial de API — ProyectaDi

## 1. Descripción general

La API de ProyectaDi será construida con FastAPI. Su función será conectar el frontend con la base de datos, aplicar reglas de negocio, validar permisos, calcular avances, gestionar presupuesto y generar reportes.

La API deberá ser modular, documentada y preparada para pruebas automatizadas.
